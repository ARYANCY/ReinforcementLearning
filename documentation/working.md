# Working Process

Each time slot follows a simple loop: observe the current state, choose an action, receive a reward, and move to the next state.

The agent learns this workflow through repeated interaction with the environment. Over time, it discovers when to transmit directly, when to harvest energy, and when to rely on backscatter or rate adaptation.

This step-by-step process is what makes the system adaptive and suitable for a research presentation on reinforcement learning in wireless networks.
