# Results Overview

The results show how the learned policy improves packet delivery in a jamming environment. After training, the agent learns to choose actions that match the current jammer state and queue status.

Q-Learning and DQN both aim to increase long-term throughput, with Q-Learning typically converging faster in this small discrete setting and DQN offering a neural approximation for scalability.

For presentation purposes, the key takeaway is that reinforcement learning can turn a hostile wireless channel into a controllable system with measurable performance gains.
