# Q-Learning Overview

Q-Learning is the tabular reinforcement learning method used in this project. It stores a Q-value for each state-action pair and updates those values from observed rewards.

Because the state space is small and discrete, Q-Learning is a strong fit here. It is simple, stable, and converges efficiently for this environment.

In this project, Q-Learning learns when to idle, transmit, harvest energy, or use backscatter and rate adaptation to maximize throughput under jamming.
