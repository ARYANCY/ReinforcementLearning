# Project Overview

This project studies reinforcement learning for anti-jamming wireless communication. The transmitter learns how to maintain reliable data delivery while operating under interference and energy constraints.

The framework combines ambient backscatter, energy harvesting, and rate adaptation in a Markov Decision Process. Two learning methods are used: tabular Q-Learning and Deep Q-Network (DQN).

The main goal is to maximize long-term throughput by letting the agent adapt to jammer behavior instead of treating the jammer only as a disturbance.
